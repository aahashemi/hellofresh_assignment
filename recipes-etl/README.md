# Recipes ETL

This project extracts recipes containing "Chilies" from a dataset and classifies their difficulty based on preparation and cooking time.

## Installation

1. Navigate to the `recipes-etl` directory:
   ```sh
   cd recipes-etl
   ```
2. Install the required dependencies:
   ```sh
   pip install -r requirements.txt
   ```
   
3. Execute the main script:
   ```sh
   python main.py
   ```

The results is saved as chilies_recipes.csv
